왜 우리가 만드는가

Behind every emitter —
we hear everything.
01
신호는 전장 인프라다

무선 스펙트럼은 현대 전장의 신경망. 모든 RF 활동이 작전 정보다 — 광대역·분산·상시 수집 없이는 전술 우위를 점할 수 없다.

02
COTS 전방 노드로 격차를 메운다

RPi5 + BladeRF / Pluto / RTL-SDR — 수백만 원으로 전방 수집 노드 구축. 고비용 플랫폼이 못 채우는 접경·해안 사각지대를 밀도 있게 배치.

03
한 분석관의 발견 = 전 체계의 자산

한 스테이션이 식별한 이미터 정보는 Signal Library를 통해 모든 스테이션에 즉시 공유. 인수인계 없이 지식이 축적된다.

04
영구 보존이 전술 우위다

놓친 Burst도, 지나간 위성 패스도 — IQ 아카이브에 남는다. 사후 재분석·증거 보전·크로스세션 상관이 가능한 유일한 구조.

06

BEWE — 3-Tier 분산 SIGINT 아키텍처

전방 수집 · C2 분석 · SIGINT 융합 — 단일 플랫폼이 세 역할을 동시에 수행한다.

07

Operational C2 View — 전방 노드 실시간 관제

3D Globe가 전술 작전도 — 전방 HOST 위치·상태를 분석관이 한눈에, 적성 위성 패스를 겹쳐서 본다.

전방 노드 실시간 마커

각 HOST가 위·경도·운용 상태를 Central로 송출 → Globe에 즉시 표시. IP 설정 없이 클릭 한 번으로 해당 스테이션 스펙트럼 접속.

적성 위성 궤적 오버레이

LEO/MEO/GEO 4계층 TLE 자동 갱신 — 中·러·北 ISR 위성 패스를 전방 노드 위치와 겹쳐 수집 기회 창 예측.

다중 스테이션 동시 관제

Globe는 작전지도, 각 전방 노드는 별도 창 — 분석관 1인이 DMZ·NLL·해안 전 구역을 동시 감시.

08

Burst 신호 회수 — 타임머신 IQ 아카이브

3초짜리 北 FH 송출도, 위성 패스 중 단발 다운링크도 — IQ 롤링 버퍼에 남아 사후 복원된다.

Max Hold / Decay 워터폴

광대역 스캔 중 모든 피크 누적 — 임무 공백 이전 RF 활동 흔적 보존. Decay로 신·구 활동 시각 분리.

Time Machine (60 sec IQ 롤링)

상시 녹화 중인 IQ 버퍼를 정지·되감기 — 이미 지나간 Burst를 스펙트럼에서 즉시 재생.

Region IQ Export

Ctrl+드래그로 시간·주파수 영역 박스 선택 → R키로 IQ 파일 즉시 추출. 증거 보전·심층 분석 워크플로우.

09

10채널 동시 복조 + Holding List

AM/FM/USB/LSB/CW 복조 10채널 병렬 운용. 분석관이 채널을 이탈해도 스켈치 타이머 유지 — 재활성 시 자동 재참여.

국가 SIGINT 누적 — 이미터 식별 DB

미션 아카이브 자동 분류
