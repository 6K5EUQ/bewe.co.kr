BEWE란?

여러 지역의 SDR 수신소를 하나의 중앙 서버로 연결하는 분산 전파 감시 플랫폼입니다. 이 페이지는 처음 사용하는 분들을 위한 단계별 안내서입니다.

시스템 구성

각 기지국(DGS)은 소프트웨어 정의 무선(SDR) 수신기를 운용하며, 중계 서버를 통해 이 웹페이지로 실시간 정보를 전달합니다.

[기지국 DGS-1] ──┐
[기지국 DGS-2] ──┤── Central Relay (port 7700) ──▶ bewe.co.kr
[기지국 DGS-3] ──┤
[기지국 DGS-4] ──┘

기지국은 한국 각지에 설치된 물리적 수신 장비입니다. 중계 서버는 모든 기지국의 상태와 채널 정보를 집계하고, 이 웹페이지는 그 데이터를 실시간으로 표시합니다.

주요 기능

📡
다채널 SDR 수신
AM · FM · DMR · AIS · ADS-B 자동 복조. 기지국당 최대 10채널 동시 운용.
🛰
실시간 위성 추적
TLE 기반 Kepler 궤도 전파. LEO/MEO/GEO 고도별 색상 구분, 궤도 경로 시각화.
🗂
IQ 히스토리 녹음
원시 IQ 스트림 장기 보관. 채널별 오디오 별도 녹음. 후처리 분석 지원.
ℹ 이 웹페이지는 읽기 전용 모니터입니다. 기지국 제어, 채널 설정, 녹음 관리는 BE_WE 운용 소프트웨어에서 수행합니다.

팝업 필드 설명

항목예시설명
Name DGS-4 (Suwon) 기지국 식별자 및 위치 표시명
Coordinates 37.25°N 127.05°E 기지국 GPS 좌표
Tier 2 수신 품질 등급. 값이 높을수록 고성능 장비
Users 3 현재 이 기지국에 접속 중인 운용자 수

💡 Freq + SR 해석 예시: Freq=433.920 MHz, SR=2.00 MS/s이면 432.920~434.920 MHz 대역을 수신 중입니다.

위성 추적

TLE(Two-Line Element) 궤도 데이터를 이용해 위성의 실시간 위치와 궤도 경로를 3D 지구본에 표시합니다.

SAT 모드 선택

  • 1
    지구본 화면 우하단 SAT 버튼 확인

    지구본 뷰에서만 나타납니다. OFF / VISUAL / STARLINK 세 가지 모드가 있습니다.

  • 2
    모드 선택

    OFF — 위성 표시 숨김
    VISUAL — 육안 식별 가능한 위성 ~1,000개 (ISS, 기상위성, 각국 군사위성 등)
    STARLINK — SpaceX Starlink 군집 ~6,000개

위성 색상 의미

위성의 색상은 궤도 고도에 따라 구분됩니다. BE_WE C++ 앱과 동일한 기준을 사용합니다.

색상궤도고도 범위대표 위성
빨강 LEO 저궤도 < 2,000 km ISS, 지구관측위성, 정찰위성
노랑 MEO 중궤도 2,000 – 35,000 km GPS, GLONASS, Galileo, BeiDou
청백 GEO 정지궤도 ~35,786 km 방송위성, 통신위성, 기상위성

궤도 경로 보기

위성 클릭 시 궤도 경로가 지구 주변에 표시됩니다

  • 1
    위성 클릭

    지구본 위의 작은 컬러 점(위성)을 클릭하면 해당 위성의 전체 궤도 경로가 지구 주위에 선으로 표시됩니다.

  • 2
    경로 숨기기

    같은 위성을 다시 클릭하거나, 빈 공간을 클릭하면 궤도 경로가 사라집니다.

📐 위성 위치는 TLE 데이터 기준 Kepler 원궤도(e=0) 근사 전파를 사용합니다. 실제 위치와 수 km 오차가 있을 수 있습니다. TLE는 1시간마다 CelesTrak에서 갱신됩니다.
